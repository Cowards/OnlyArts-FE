<br> `Guest`

<br> `Audiences`:
<br> &#8194;&#8194; **Purchasing premium artworks**
<br> &#8194;&#8194; **Sendind unique artwork request to** `Creator`
<br> &#8194;&#8194; **Sendind** `System` **report**
<br> &#8194;&#8194; **Sendind artworks report**

<br> &#8194;&#8194; **Sendind artworks report**
